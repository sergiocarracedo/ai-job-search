# /upskill — Skill Gap Analysis and Learning Plan

Use the **job-scraper** skill to identify gaps between your profile and tracked job postings:

**`/upskill`** — Run in aggregate mode: analyses all jobs in `job_search_tracker.csv`

**`/upskill <URL>`** — Run in targeted mode: analyses a single job posting from the URL

Invoke the **upskill** skill now with `$ARGUMENTS` as the argument.
